 # Roast Toast Coast Cafe ☕🌊

A modern, responsive café menu website built using **HTML, CSS, and JavaScript**. The project features a stylish dark theme, animated accordion menu sections, and a fully responsive layout that provides a smooth user experience across desktop and mobile devices.

---

## 🚀 Features

* Modern café landing page
* Responsive design for desktop and mobile
* Animated accordion menu categories
* Smooth CSS transitions
* Glassmorphism hero section
* Beautiful dark theme with gold accents
* Accessibility support using ARIA attributes
* Interactive JavaScript menu toggle

## ⚡ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/roast-toast-coast-cafe.git
```

2. Open the project folder

```bash
cd roast-toast-coast-cafe
```

3. Open **Menu.html** in your preferred web browser.

No installation or dependencies are required.
